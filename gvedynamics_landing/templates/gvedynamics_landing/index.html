О компании

GVE Dynamics — это IT-компания, разрабатывающая решения для анализа действий пилотов и проактивного мониторинга параметров полёта. Наша цель — сделать авиацию безопаснее с помощью данных и технологий будущего.

Что мы делаем

Программа LOSA

Аудитор заполняет таблицы отчета в приложении и при наличии интернета данные сохраняются в базу данных на сервере, который выводит автоматизированный анализ данных для Ваших должностных лиц. подробнее...

Тренажерная подготовка

Инструктор тренажера оценивает пилотов при помощи выпадающих списков, что экономит время на записи в задание на тренировку. После завершения сессии формируется бланк тренировки для печати, готовый для подписи, с оценками и замечаниями. Данные отправляются на сервер для дальнейшего анализа и статистики. Формируются графики компетенций пилота. подробнее...

Line Check - Летная проверка в воздухе

Пилот - инструктор оценивает элементы в офлайн приложении. Формирует задание на тренировку для подписи. При наличии интернета приложение отправляет данные на сервер. Данные добавляются в графики компетенций пилота. подробнее...

Анализ по средствам объективного контроля

По Вашему выбору, тех параметров, которые требуется мониторить, делаем выборку данных из файлов расшифровки полетов. Данные добавляются в графики компетенций пилота. подробнее...
